# NDVI Analysis using Sentinel-2 Satellite Imagery

## Overview
This project explores NDVI (Normalized Difference Vegetation Index) analysis using Sentinel-2 satellite imagery in Google Earth Engine. The objective was to understand vegetation patterns and visualize spatial variation using remote sensing data at 10m resolution.

## Objective
To explore how satellite imagery can be used to analyze vegetation health and distribution through NDVI-based mapping.

## Methodology
The workflow involved:

- Accessing Sentinel-2 imagery in Google Earth Engine
- Applying cloud masking to improve image quality
- Computing NDVI values using near-infrared (NIR) and red spectral bands
- Visualizing vegetation patterns spatially

## Tools & Technologies
- Python
- Google Earth Engine
- Google Colab
- Sentinel-2 Satellite Imagery
